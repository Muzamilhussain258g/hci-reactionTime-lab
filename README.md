# ⚡ Reaction Time Lab (HCI Project)

A simple and interactive **Reaction Time Testing Web App** built using **Vanilla JavaScript** as part of an HCI (Human-Computer Interaction) lab.

---

## 🎯 Project Overview

This project measures a user's **reaction time** by displaying random stimuli (letters or numbers) and recording how quickly the user responds using keyboard or buttons.

It demonstrates key HCI concepts such as:
- User input handling
- Immediate feedback
- Response time measurement
- Error tracking

---

## 🧠 How It Works

- A random **letter (A–Z)** or **number (0–9)** appears on screen
- User must respond:
  - Press **A** → if it's a letter
  - Press **L** → if it's a number
- System records:
  - Reaction time (ms)
  - Correct responses
  - Mistakes

---

## 🎮 Game Rules

- Total Trials: **10**
- Maximum Mistakes: **3**
- Game ends when:
  - All trials are completed ✅
  - OR mistakes reach 3 ❌

---

## 📊 Features

- ⏱ Real-time reaction time tracking
- 📈 Average response time calculation
- ✅ Instant visual feedback
- ❌ Error detection system
- 🎯 Keyboard + Button input support
- 🔁 Automatic trial progression

---
